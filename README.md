# Open Fit

**Your health, your data, your way.**

The definitive free open-source fitness app for iOS and Android. Workouts, nutrition, glucose tracking, body metrics, and more — all in one synthwave-styled package.

## Features (Planned)

- **Workouts** — Exercise library, custom routines, logging, PRs, rest timers
- **Nutrition** — Macro tracking, meal logging, barcode scanning, recipes
- **Health** — Glucose logging (diabetic-friendly), blood pressure, heart rate
- **Body Metrics** — Weight, measurements, body fat %, progress photos
- **Analytics** — Dashboard, charts, streaks, data export

## Philosophy

- **Free & Open Source** — GPL-3.0 licensed, forever free
- **Offline-First** — Your data stays on your device
- **No Ads, No Tracking** — Your health data is yours alone
- **Synthwave Aesthetic** — Retro-futuristic design with neon flair

## Tech Stack

| Component | Technology |
|-----------|------------|
| Framework | Flutter 3.x |
| Language | Dart |
| State | Riverpod |
| Database | Isar |
| Charts | fl_chart |

## Getting Started

```bash
# Clone the repo
git clone https://github.com/synth/open-fit.git
cd open-fit

# Install dependencies
flutter pub get

# Run on device
flutter run
```
